# 🧠 AI Resume Analyzer

An intelligent **AI-powered Resume Analyzer** that evaluates resumes using **Natural Language Processing (NLP)** and **Generative AI**.
It helps users improve their resumes by providing **skill extraction, job matching, and smart feedback**.

---

## 🚀 Features

* 📄 Upload Resume (PDF, DOCX, TXT)
* 🧠 NLP-based Keyword & Skill Extraction
* 🎯 Job Description Matching
* 📊 Resume Match Score (%)
* ❗ Missing Skills Detection
* 🤖 AI-based Resume Feedback
* 💻 Simple & Clean Web Interface

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python
* **Libraries:** spaCy, NLP tools
* **AI:** Generative AI APIs (optional)

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/Resume-Analyzer.git
cd Resume-Analyzer
```

---

### 2️⃣ Install Dependencies

```
pip install -r requirements.txt
```

---

### 3️⃣ Install NLP Model

```
python -m spacy download en_core_web_sm
```

---

### 4️⃣ Run the Project

#### ▶️ Frontend Only

```
python -m http.server 5500
```

Open:

```
http://localhost:5500/resume_analyzer_app.html
```

#### ▶️ With Backend

```
python resume_analyzer.py
```

## 📊 How It Works

1. Upload or paste resume
2. Extract text from file
3. Analyze using NLP
4. Identify skills & keywords
5. Compare with job description
6. Generate match score & feedback
